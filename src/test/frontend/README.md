# Frontend Tests

## data-test-id и data-test-group на сайте

Сайт использует атрибуты для стабильных селекторов в автотестах:

- **data-test-id** — уникальный идентификатор элемента. Используется вместо классов/вёрстки, чтобы тесты не ломались при изменении стилей. Примеры: `nav-link-products`, `main-image`, `products-title`.

- **data-test-group** — общая метка для группы однотипных элементов. Позволяет находить все элементы группы и обращаться к ним по индексу или фильтру. Примеры: `nav-link`, `product-card`, `cart-item`.
